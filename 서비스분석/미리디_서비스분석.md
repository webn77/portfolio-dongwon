## 2. 사용자층 분석

### 공식 데이터
- 가입자 46% 학생 | 여성 74% | 10~20대 58%
- 실제 결제 유저: 소상공인·마케터·크리에이터
- 기업 고객: 2,800개+ (한국 사업체 700만 대비 0.04%)

### 세그먼트 재정의 (행동 기반)

#### 🎤 세그먼트 A — 학생 = K팝 팬 크리에이터
- **재정의:** 수행평가 유저가 아닌 팬덤 굿즈 제작자
- **행동:** 포토카드·생일카페 현수막·팬북 제작
- **결제 트리거:** 시즌 단기 패스 (현재 없음)
- **시장:** K팝 4대 엔터사 굿즈 매출 1조원 돌파 예상
- **경쟁자:** 포토몬, 프린팅팅 → 비즈하우스로 원스톱 가능

#### 💼 세그먼트 B — 개인사업자·소상공인
- **핵심:** 디자인 → 인쇄 원스톱이 필요한 사람
- **결제 트리거:** 배경제거 소진, AI 크레딧 소진, 비즈하우스 인쇄 주문 후
- **현재 문제:** 비즈하우스 별도 로그인 → 마찰 발생
- **비즈하우스 고객:** 카페·식당 21%, 커머스 19%, 교육 15%

#### 🏢 세그먼트 C — B2B 인사·총무팀
- **타겟:** 인사팀·총무팀 (정기 반복 발주)
- **발주 항목:** 신입사원 웰컴키트 (사원증·명함·메시지카드·텀블러), 명함, 현수막, 판촉물
- **결제 동기:** Adobe CC + 외주 비용 절감
  - Adobe CC: 1인당 월 6~8만원
  - 외주 디자인: 월 30~100만원
  - 미리캔버스 Enterprise: 1인당 월 33,000원

## 3. 서비스 구조 분석

### 요금제 구조
| 요금제 | 가격 | 핵심 기능 |
|---|---|---|
| 무료 | 0원 | 5만개 템플릿, 기본 편집, 1GB |
| Pro | 월 14,900원 | 프리미엄 무제한, 배경제거, AI 5배, 브랜드키트, 피드백·승인 |
| Enterprise | 월 33,000원+ | Pro 전체 + SSO/API + 전담매니저 + 무기한 복구 |
| Edu+ | 별도 | 교사·학생 전용 |

### 핵심 Lock-in 포인트
- **브랜드키트 첫 세팅 순간** = 가장 강력한 Lock-in
  - 로고 업로드 → 사업자 신호
  - 팀 색상 설정 → 팀 있음
  - 반복 디자인 → 헤비 유저

### 앱의 치명적 한계
- 배경제거·AI 기능 앱 미지원 (핵심 전환 트리거가 없음)
- 앱 구독 시 스토어 수수료로 웹보다 가격 비쌈
- 모바일 에디터 2.0은 2026년 상반기 출시 예정

### FAQ 인사이트
- **저작권 질문 압도적 1위** → 상업적 사용 유저(소상공인)가 많다는 신호
- 요금제 결제 구조 혼란 (워크스페이스 단위 결제)
- 자동 결제 불안 → 마지막 허들

## 4. 수익 구조 분석

### 전환율 문제
- 현재: 0.6% (가입자 1,600만 → 유료 ~10만명 추정)
- Canva 글로벌: 9~10%
- 격차 원인: 세그먼트 미분류 + 전환 트리거 단일화 (왕관 팝업만)

### 비즈하우스 연결 구조 (강점)
```
미리캔버스 에디터 → [인쇄물 제작] 버튼
→ 인쇄물 유형 선택 → 가이드 자동 적용 (규격 자동 세팅)
→ 옵션 선택 → 장바구니 → 비즈하우스 결제 → 배송
```
- Canva는 이 구조가 없음 → 미리디만의 독보적 강점
- 단, 별도 로그인 조건 (동일 이메일) → 마찰 포인트

## 5. 핵심 인사이트

### 미디디가 Canva를 따라가면 영원히 2등
| 지표 | Canva | 미리캔버스 |
|---|---|---|
| 글로벌 가입자 | 1억 7천만 | 1,600만 |
| 유료 전환율 | 9~10% | 0.6% |
| 실물 인쇄 연동 | ❌ | ✅ 비즈하우스 |
| 모바일 완성도 | ✅ | △ (기능 미완성) |

**Canva는 화면을 만들어줌. 미리디는 실물을 만들어줌.**

### 구조적 문제 3가지
1. **세그먼트 미분류** — 1,600만 유저에게 왕관 팝업 하나로 전환 시도
2. **서비스 간 연결 단절** — 미리캔버스 ↔ 비즈하우스 별도 로그인
3. **앱 기능 불완전** — 핵심 전환 트리거(배경제거·AI)가 앱에 없음

### 반복되는 패턴
- 많이 쓰는 유저 ≠ 돈 내는 유저 (학생 46%지만 결제는 소상공인)
- FAQ 순서 = Pain Point 순서 (저작권 질문 1위 = 소상공인이 많다)
- Lock-in 시점 미활용 (브랜드키트 세팅 순간이 최적 전환 타이밍)

## 6. AI 제안 — MCP 기반 차별화

### 핵심 슬로건
> **"말하면 만들어지고, 만들면 배송된다"**

### MCP + LLM 연동 시나리오

**인사·총무팀:**
```
Slack: "김철수 대리 입사, 명함 100장 주문해줘"
→ LLM이 미리캔버스 MCP 호출
→ 브랜드키트 자동 적용 → 명함 생성
→ 비즈하우스 자동 주문 → 3일 후 배송
```

**소상공인:**
```
"이번 주 벚꽃 이벤트 포스터 만들어줘"
→ 브랜드키트 자동 적용 → 3종 생성
→ 선택 → 비즈하우스 인쇄 주문 → 내일 수령
```

**K팝 팬:**
```
"○○ 생일카페 굿즈 세트 만들어줘"
→ AI 배경 제거 → 포토카드 규격 자동 세팅
→ 비즈하우스 패키지 주문 원클릭
```

### 세그먼트별 제안 요약
| 세그먼트 | 핵심 제안 | 기대 효과 |
|---|---|---|
| K팝 팬·학생 | AI 포토카드 자동 제작 + 비즈하우스 소량 인쇄 + 디자인허브 수익화 | 포토몬·프린팅팅 대체 |
| 소상공인 | SSO 통합 (별도 로그인 제거) + AI 작업 감지 → 인쇄 원클릭 | 비즈하우스 전환율 즉시 개선 |
| B2B 인사·총무 | MCP + HR시스템 연동 + 웰컴키트 자동화 + Adobe ROI 계산기 | Enterprise 반복 구매 Lock-in |

### 3단계 로드맵
- **단기 (0~3개월):** SSO 통합, 배경제거 소진 트리거 A/B 테스트
- **중기 (3~12개월):** K팝 팬 AI 제작 MVP, 인사팀 웰컴키트 패키지
- **장기 (1년+):** MCP 서버 공개, HR시스템 연동, 글로벌 K팝 팬 확장
