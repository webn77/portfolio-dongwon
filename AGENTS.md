# AGENTS.md

이 폴더는 Codex 실행용 얇은 인터페이스다.

## 우선 참조
- 공통 규칙: 상위 `AGENTS.md`
- 로컬 상세: `CLAUDE.md`
- 충돌 시 더 가까운 문서를 우선 적용한다

## Codex 작업 규칙
- 작업 전에 `CLAUDE.md`에서 경로, 산출물, 금지사항을 먼저 확인한다
- 수정과 생성은 요청 범위 안에서만 수행한다
- 완료 전에는 테스트 또는 가능한 최소 검증을 수행한다
- 기록이 필요한 작업이면 로컬 규칙에 맞춰 문서와 작업내역을 반영한다
- 새 하위 작업 폴더를 만들면 같은 패턴으로 `AGENTS.md`를 둔다
